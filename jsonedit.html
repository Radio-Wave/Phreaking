GitHub Configuration

  1. On GitHub, click your avatar → Settings
  2. Scroll to Developer settings (bottom of left sidebar)
  3. Choose Fine-grained tokens → Generate new token
  4. Set Resource owner to radio-wave and pick the phreaking repo under Only select repositories
  5. Under Repository permissions set Contents to Read and write
  6. Click Generate token, copy it, paste it above

For a classic token: use github.com/settings/tokens/new and tick the repo scope.

Token needs Contents: Read & write (fine-grained) or repo scope (classic).
Stored only in your browser's localStorage — never sent anywhere except GitHub.
